Problem 1

This problem is intended to emphasize variable types and introduce coercion; some awareness of how R treats numeric, character, and factor variables is necessary for working with these data types in practice.

Create a data frame comprised of:

a random sample of size 10 from a standard Normal distribution a logical vector indicating whether elements of the sample are greater than 0 a character vector of length 10 a factor vector of length 10, with 3 different factor levels

set.seed(123)

df = tibble(
  sample = rnorm(10),
  vec_logical = sample > 0,
  vec_char = c('A','B','C','D','E','F','G','H','I','J'), 
  vec_factor = factor(c('1','2','3','1','2','3','1','2','3','1'))
)

Try to take the mean of each variable in your dataframe. What works and what doesnt? Hint: to take the mean of a variable in a dataframe, you need to pull the variable out of the dataframe. Try loading the tidyverse and using the pull function.

mean(pull(df, var = sample))
## [1] 0.07462564
mean(pull(df, var = vec_logical))
## [1] 0.5
mean(pull(df, var = vec_char))
## Warning in mean.default(pull(df, var = vec_char)): argument is not numeric or
## logical: returning NA
## [1] NA
mean(pull(df, var = vec_factor))
## Warning in mean.default(pull(df, var = vec_factor)): argument is not numeric or
## logical: returning NA
## [1] NA

We are able to take the mean of numeric and logical variable, but when taking the mean of logical variable, false is converted to 0 and true is converted to 1 and the mean of it is the sum of 0 and 1 over the length of the vector. We are not able to take the mean of character vector or the factor vector.

In some cases, you can explicitly convert variables from one type to another. Write a code chunk that applies the as.numeric function to the logical, character, and factor variables (please show this chunk but not the output). What happens, and why? Does this help explain what happens when you try to take the mean?

as.numeric(pull(df, var = vec_logical))
as.numeric(pull(df, var = vec_char))
as.numeric(pull(df, var = vec_factor))

Logical variable was able to be converted to numeric variable which false was converted to 1 and true was converted to 0. Character variable was not able to be converted to numeric variable and we got NA for all the entries because it doesnt make sense to have numbers represent character variable. Factor variable was able to be converted to numeric variable indicating three levels. This helps explain why when we try to take the mean we were able to get result for logical variable but not for either character variable or factor variable.
